# ppass - 跨网络P2P文件管理系统

> 把闲置电脑变成私有云，无需公网IP，无需复杂配置

## 🎯 目标

- **存储端**：选择本地文件夹 → 生成配对码 → 等待连接
- **访问端**：输入配对码/扫码 → 连接 → 浏览/下载文件

## 🛠 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vanilla JS + Vite |
| P2P传输 | WebRTC (PeerJS) |
| 文件API | File System Access API |
| 传输协议 | DataChannel (可靠模式) |

## 🚀 快速开始

### 开发模式

```bash
# 安装依赖
cd ppass
npm install
cd src/web && npm install

# 启动
npm run dev
```

### 使用

1. **存储端**：打开网页 → 选择"作为存储端" → 选择文件夹 → 分享Peer ID
2. **访问端**：打开网页 → 选择"作为访问端" → 输入Peer ID → 连接 → 浏览文件
